# 🎬 Movie App

A modern mobile movie application built with **React Native** and **Expo**, featuring a clean UI, smooth navigation, and scalable architecture.

---

## ✨ Features
- Browse movies with a clean and responsive UI
- Bottom tab navigation
- Modern routing using Expo Router
- Haptic feedback for better UX
- Optimized images and splash screen
- Cross-platform (Android, iOS, Web)

---

## 🛠 Tech Stack
- **React Native**
- **Expo**
- **Expo Router**
- **TypeScript**
- **NativeWind (Tailwind CSS)**
- **React Navigation**
- **Appwrite** (Backend & Authentication)
- **Expo Image**
- **Expo Haptics**

---

## 📱 Platforms
- ✅ Android  
- ✅ iOS  
- ✅ Web
